# The Gao Family Journey

The official website of the Gao family — documenting the learning journey of Roy, Chica, and Sean Gao, three elementary school students in Shenzhen, China, along with their Shetland Sheepdog Xiaodai.

## Website

[Live Site](https://tyz6qfwiksuew.ok.kimi.link)

## Features

- **Bilingual Support** — Toggle between English and Chinese
- **Single-Page Layout** — Smooth scrolling with GSAP animations
- **Responsive Design** — Works on desktop, tablet, and mobile
- **Custom Cursor** — Interactive cursor with hover states
- **Scroll Animations** — Parallax effects and section transitions

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS 3
- GSAP with ScrollTrigger
- Lenis Smooth Scroll

## Project Structure

```
src/
  sections/       — Page sections (Hero, About, Projects, Milestones, Contact, Footer)
  hooks/          — Custom hooks (Language, Lenis, Cursor)
  i18n.ts         — Bilingual translations (EN/ZH)
  config.ts       — Site configuration
public/
  images/         — Photos including real Xiaodai pictures
```
